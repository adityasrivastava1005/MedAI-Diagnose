# MedAI Diagnose
### CNN-Based Medical Diagnosis Pipeline Using PEPA (Patient-Evidence-Prior Attention)

---

## Overview

MedAI Diagnose is a multimodal deep learning system for preliminary disease prediction built with **PyTorch**. It accepts two inputs:

1. **Symptom text** — primary input, always required
2. **Medical image** — optional (X-ray, skin photo, scan, wound)

Both inputs are processed independently. The **PEPA attention gate** (Patient-Evidence-Prior Attention) then learns to combine them adaptively — trusting whichever modality provides stronger signal per patient.

```
Patient types:  "fever, cough, chest pain"
Patient uploads: chest_xray.jpg  (optional)

        ↓                     ↓
  NLP Encoder           CNN Encoder
  TF-IDF + Dense        EfficientNetB0
  f_txt ∈ R¹²⁸          f_img ∈ R¹²⁸

        └──────────┬───────────┘
                   ↓
             PEPA Gate
        α = σ(W·[f_img ; f_txt])
        fused = α⊙f_img + (1-α)⊙f_txt

                   ↓
        Dense(256) → Dropout → Softmax

                   ↓
    1. Pneumonia         84%
    2. Bronchitis          9%
    3. COVID-19            4%
    Specialist: Pulmonologist
    Emergency: No
```

## Tech Stack

| Component | Technology |
|---|---|
| Deep Learning Framework | PyTorch cu128 |
| CNN Backbone | EfficientNetB0 (ImageNet pretrained) |
| NLP Encoding | TF-IDF + Dense layers |
| Fusion | PEPA attention gate (custom) |
| Web Framework | Flask |
| ML Baseline | scikit-learn Random Forest |

## Ablation Study — 7 Variants

| Variant | Configuration | Top-1 | Top-3 | F1 Score |
|---|---|---|---|---|
| A1 | Random Forest (baseline) | 80.31% | 92.98% | 0.8116 |
| A2 | CNN encoder only (image-only) | 67.8% | 86.3% | 0.6544 |
| A3 | NLP encoder only | 85.27% | 96.5% | 0.8546 |
| A4 | CNN + NLP concat fusion | 86.7% | 96.81% | 0.8677 |
| A5 | CNN + NLP mean fusion | 86.75% | 96.98% | 0.8691 |
| A6 | CNN + NLP + PEPA (no dropout) | 86.95% | 97.14% | 0.8696 |
| **A7** | **CNN + NLP + PEPA (proposed)** | **87.0%** | **97.14%** | **0.8707** |

## Quick Start

### 1. Install PyTorch with CUDA 12.8

```cmd
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu128
```

### 2. Install remaining dependencies

```cmd
pip install -r requirements.txt
```

### 3. Verify GPU

```python
import torch
print(torch.cuda.is_available())         # True
print(torch.cuda.get_device_name(0))     # NVIDIA GeForce RTX 5050
```

### 4. Download datasets

```cmd
kaggle datasets download -d tawsifurrahman/covid19-radiography-database
kaggle datasets download -d masoudnickparvar/brain-tumor-mri-dataset
# ... (see clean_image_datasets.ipynb for full list)

# DDXPlus text dataset (no login needed):
# https://figshare.com/articles/dataset/DDXPlus_Dataset_English_/22687585
```

### 5. Run notebooks in order

```
1. notebooks/clean_text_datasets.ipynb
2. notebooks/clean_image_datasets.ipynb
3. notebooks/train.ipynb              ← 3-4 hours on RTX 5050
4. notebooks/evaluate.ipynb
```

### 6. Start Flask web app

```cmd
cd webapp
python app.py
```

Open: http://127.0.0.1:5000

### 7. Health check

```
http://127.0.0.1:5000/health
```

Returns:
```json
{
  "status": "ok",
  "device": "cuda",
  "gpu_name": "NVIDIA GeForce RTX 5050",
  "vram_total": "8.0 GB",
  "disease_classes": 443,
  "model_loaded": true
}
```
